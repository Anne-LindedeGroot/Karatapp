# Database Scripts

This directory contains all SQL scripts and database-related files:

## Supabase Tables
- `supabase_forum_tables.sql` - Forum tables creation and setup
- `supabase_katas_table.sql` - Katas table creation and setup
- `supabase_host_fix.sql` - Host configuration fixes
- `supabase_permission_cleanup.sql` - Permission cleanup scripts

## Migrations
- `modify_existing_katas_table.sql` - Migration script for modifying existing Katas table

## Usage
These scripts should be run in your Supabase SQL editor or via the Supabase CLI to set up and maintain your database schema.
